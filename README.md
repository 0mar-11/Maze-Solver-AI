# 🤖 Maze Solver AI - BFS Pathfinding Visualization

An interactive GUI application that solves mazes using the **Breadth-First Search (BFS)** algorithm with real-time visualization built with Python and Tkinter.

---

## ✨ Features

- 🎲 **Random Maze Generation** - Creates unique mazes with customizable obstacles
- 🧭 **BFS Pathfinding Algorithm** - Guarantees the shortest path solution
- 🎨 **Visual Path Display** - Color-coded gradient showing the solution path
- 🖱️ **Interactive GUI** - User-friendly interface with Tkinter
- 📊 **Step-by-Step Visualization** - Watch the algorithm explore the maze
- 🎯 **Clear Start/End Points** - Green (start) and Red (end) markers

---

## 🛠️ Tech Stack

- **Language:** Python 3.x
- **GUI Framework:** Tkinter
- **Algorithm:** Breadth-First Search (BFS)
- **Data Structures:** Queue (deque), Sets, 2D Arrays
- **Libraries:** `collections.deque`, `random`, `tkinter`

---

## 📋 Prerequisites

- Python 3.6 or higher
- Tkinter (usually pre-installed with Python)

---

## 🚀 Installation & Usage

git clone https://github.com/0mar-11/maze-solver-ai.git
